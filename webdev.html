Web development

August 20, 2022 by Arbaaz

If you're looking for a way to engage with potential customers and clients, web development is a great way to start. Web developers use the code below to create websites that look beautiful on any device and work in any browser.

HTML

HTML is the language used to create web pages. It's short for HyperText Markup Language, and it's a markup language.

HTML allows you to define the structure of your web page, including:

  • The heading section at the top of every page (h1)
  • Paragraphs (p) and other elements such as lists and tables that compose your content section(s).

CSS

CSS stands for Cascading Style Sheets and is a part of HTML5. It allows you to style your web page to look pretty, professional and even fun! The best way to learn CSS is by playing around with it. I like to use the [Firefox] browser because it's free, easy-to-use and has tons of tutorials on how to use CSS (and other languages).

Once you've got that down, here's what you need to know about using CSS:

  • It's a style sheet language used for defining how elements look on a webpage; for example, it can define colors and fonts.
  • It defines rules for formatting HTML elements like text or images so that they look great when rendered in different browsers.

JavaScript

JavaScript is a programming language that can be used to make webpages interactive. It’s also used to make webpages more dynamic, responsive and user-friendly.

With javascript you will be able to create an interactive webpage or site by using some pre-written code that responds to user actions. This is called “scripting” or “programming” in general terms with the word “script” referring specifically to written programs used on computers (hence why it can be referred as such).

Build a Simple website using the code below:

            <!doctype html>
                <html lang="en">
		    <head>
		        <title>Demo.</title>
		    </head>
		    <body>
		        <h1>Sample Wesbite</h1>
		        <h2>My first page.</h2>
		    </body>
	        </html>

Conclusion

I hope this post was helpful in explaining the basics of web development.

The next step is to get started on your own website and see how it looks!
